If you want a primarily application–based class and have some data analysis coding experience, ESE 102 is for you. This year there were no exams, which was really nice so I could get exposure to a lot of different statistical techniques with low pressure. However, I think you'll get a lot more out of the course than I did if you know some Stat 110 and 111 material. I took Stat 110 concurrently with the class and I only felt like I had a grasp of the theory behind most concepts a bit more than halfway through the semester.

I'm an AM major and want to work in climate on quantitative modeling/technology. Initially, I figured this class would be a great way to introduce me to potential intersections of those fields, but Roger's work surrounds earth science (think rocks/geology/paleoclimate) so we didn't do as much application to climate as I would have liked. The course also didn't really ensure that everyone had a strong grasp on the theory, so I think the better choice for me would have been to take Stat 111 instead of (rather than in addition to) this class.
Good class. Roger, Esther, and Ethan are lovely!!! Sometimes lecture can feel a little bit foreign and un–explained, but the course staff is there to help. Projects are really cool and a great opportunity to apply what you're learning.
This class is hands down one of the best classes that I have taken at Harvard. I am very interested in Environmental Science and Data Science so this class was a perfect mix of the two. I learned so so many useful tools for statistical inference that I will continue to use in my lab's research, future class projects, and all other data analysis that I do in the future. Roger is so so nice and really loves the topics covered in class. The class is full of environmental and stats kids which is a great, welcoming combo. I would 100% recommend this class to anyone for the tools that it gives you to analyze any kind of data to understand the world around you. Very applied and useful.
I have coded a lot before this class which was very very helpful in the psets, BCAs, and projects. I would recommend that you come into this class with some coding background to get the most out of it. If not, be prepared to learn a lot about coding and put in the time to do so. You will have a lot of help from friends and the TFs.

Also, the projects are open ended, so it is hard to find something manageable, but also interesting.

I would recommend this class to anyone who has a little bit of a stat or python background because it will make your life so much easier. Otherwise I might stick to a class more like stat 100 because I think that has more structure to teach coding and really a solid background.

This is an awesome choice to fill your distributional requirement or department requirement for math/statistics! There are lots of group projects, very few problem sets and some assignments that are graded for completion. The lectures are the weakest part of the class, but despite that I still got a lot out of it because we had to apply the material to projects throughout the course. That being said, I think I would have gotten more out of the class if I had some prior statistics background. Nevertheless, if you're interested in research and how to use statistics, definitely consider this course!
