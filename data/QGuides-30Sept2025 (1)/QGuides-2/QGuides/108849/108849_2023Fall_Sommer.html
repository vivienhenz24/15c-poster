This class is heavy. It's worth it—it improved my ability to read in Spanish and got me to read some new books (which I always appreciate)—but the first half of the class is a heavy lift. Highly recommend to anyone interested in engaging with really unique works that deal with issues like race and agriculture, and improving their Spanish in an inviting environment. Professor Sommer is eager to help and excited about the content. My number one suggestion is to talk in class! Raise your own points. You can drive the conversation as you'd like. The class is very much a democracy.

This class was daunting at first, but I was happy by the end by how much my Spanish had improved. The main takeaways that I had were to do with the idea of the "Caribbean identity" and how it is so challenging to define and why. There is a lot of reading, mostly in Spanish, so I would recommend complete fluency and an excitement to discuss the texts offered before enrolling
This class is challenging but fair. There is a lot of reading but relatively little work outside the readings. The class itself is entirely discussion–based and was enjoyable. The only issue with this class is that there is very little feedback on assignments, so you do not know how well you are doing. Take this class if you are confident in your Spanish reading and writing abilities and want to learn more about the subject matter. Read the reading list and see if you are interested in the books, because the class is entirely centered around the weekly readings.
Take this class! It's not super easy but you will learn a lot and will become a better student.
