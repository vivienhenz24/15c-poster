It's really incredible how much you learn in one semester, but this course is very draining so I would only recommend it if you're dedicated to continuing with Arabic. I would 1000% recommend taking Richard Cozzens section if you can and Abdulgaleel is the best TF, but if you have a specific interest in a dialect then try to find an instructor that teaches that one (Cozzens teaches the Levant dialect) You will begin to loose faith about mid way through the semester when you've finished learning the alphabet but it'll all come together in the end!!! Seriously it's a very rewarding course, but attendance is weighted VERY highly so be careful if youre taking a 9am section.
This is a wonderful class. The professor is amazing and he is so caring and understanding if you are ever sick. I will say that if you don't have any past experience in learning Arabic then you should put in extra effort in studying each night. Build up a daily review routine since the class does start going a bit fast after the first couple of weeks, and you learn a lot of words every week. There are about six quizzes, but they aren't too bad if you know the class material. I think this is an amazing introductory class.
If you have any interest in Arabic, I highly recommend this course. I came in not speaking a lick and came out knowing how to read and converse on a very basic level, which I never could've dreamed of doing without this course. Ustaaz Richard is truly the best, not only as an Arabic AA professor but as an educator more broadly. He will make sure that this course is manageable and also meaningful. You will learn a lot if you do your work (which is totally doable), and you will get a lot out of it in return. Cannot recommend enough.
