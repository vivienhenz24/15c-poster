What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
Take this if you want to write a thesis/ feel like your stat experience has been a little disjointed or if you feel disconnected from the department
If you want a class that allows you to apply what you've learned in statistics and cs classes, then this is a good option. Some weeks this class has very little work, while for others, Stat 98 will be your most time consuming class.
This course is great in theory. However, there are so many pitfalls in practice that make it not worth taking.You get a lot of practice presenting about and writing about statistics, but it's hard to feel like you're doing well or improving because the feedback is extremely delayed or nonexistent. I took this class because I thought it might help me prepare for writing a thesis, but I think there are better ways of achieving this goal other than this class. The only useful takeaway from this class was learning how to use beamer (presentations with LaTeX)
This is a great class that gives you a lot of flexibility to pursue topics of your interest, but you should be prepared to put in a lot of work. The grading is quite critical and the demands are very high, but I think it can be a very rewarding experience.
This is not a reading class! The emphasis is not on familiarizing yourself with statistical research and you do not regularly critique/review published papers. The primary project is developing your own three projects.
