Take this class if you are genuinely interested and invested in the topic. It does involve a fair bit of research and writing and the assessment can feel a bit opaque at times. Make sure you read the syllabus ahead of time and are okay with pop quizzes and a 50+ page final paper. The final project is a great opportunity to do a deep dive into water resources in a country you're curious about.
This class is a great class that teaches you a lot of skills. It can be a lot of work during some weeks, but luckily the final project is easy because you do part of it each week in the semester so it is more about compiling than writing. I recommend this course and Ken is an engaging lecturer!
This class is really hard. Expect to write a five page paper every week, on top of separate weekly homework assignments. There are also pop quizzes roughly every two weeks throughout the semester, which can be really stressful. However, this class is ultimately really rewarding because of the term project, where your weekly papers add up to a 50–75 page paper. It's definitely a cool class, but I would really only recommend it if the rest of your schedule is really light.
