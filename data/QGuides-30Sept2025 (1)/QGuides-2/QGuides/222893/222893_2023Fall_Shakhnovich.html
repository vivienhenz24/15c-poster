What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
Prof. Shakhnovich is an excellent lecturer, he begins the class with lectures that allow every student to be on the same level when it comes to the course material. The assignments are very relevant and you get to learn a lot about protein folding and phase separation.
You should take this course if you will work on protein engineering. You can learn the basics of the field which is necessary to do research on protein folding and liquid–liquid phase separation.
This course both teaches stat mech in a biophysical setting, as well as public speaking. I learned a lot about writing proposals and preparing discussions for the class.
This class is purely for your personal interest in the topics covered. If you find that you aren't interested in the topics listed in the syllabus, then the class is not for you.
