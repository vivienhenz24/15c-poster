You need to have a strong background in statistics, algorithm, and genomics to understand all the contents in this class.
Take it! It's worth it.
Learn Math
This is a class one should take if one has a strong background in CS and/or statistics and wishes to get a better sense of how these topics can be used to ask and answer foundational scientific questions. However, it should not be used as a class to learn these CS/stats techniques–––for that, choose a more technically rigorous class (e.g. Stat 111, CS 25, etc.)
This is an excellent choice if you're looking for a broad survey of mathematical/computational techniques for analyzing genetic data. For people with some mathematical/computational background, the workload is very reasonable and so lets you balance it with research and other classes.
