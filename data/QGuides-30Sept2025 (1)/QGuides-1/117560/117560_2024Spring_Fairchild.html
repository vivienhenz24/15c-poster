It's a very manageable class that provides you with a pretty basic, but very solid understanding of research methods. Don't let the mid–term surprise you –– it is very fair, covering exactly the information covered in class up to that point, but it does require you to thoroughly know the material. This is a pleasant, relatively easy class covering topics that you will almost certainly have some passing familiarity with as a layperson, so it is easy to get lulled into a false sense of security and only realize halfway through the exam that you don't actually know how to articulate what 'generalizability' is.
Take this class if you want to learn more about research. I wasn't that interested in research before this class, but now I am. This class teaches you about the multiple ways research takes form and really helps you understand it. The assignments are very doable, and the readings are very interesting and engaging.

this is a requirement for sociology concentrators. There are six lab assignments throughout the course that essentially ask you to practice the methods learned in class by proposing how you might conduct them in a research project of your own. If you are thinking about a thesis, this class gets you prepared to some degree in that it shows you part of what is needed to make that happen. The lectures allow room for a lot of participation, which helps with a bit of buy in and helps make things easier to handle. If you need any kind of accommodations, make sure to stay in close communication with your tF, because they will be your biggest advocate as things come up.
