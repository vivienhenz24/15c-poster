I was genuinely disappointed to miss class when I had to, because it was so much fun. Great class! No prereq is necessary if you have some background with CS or formal logic. Professor Sag really cares about the students, and the topic is very interesting for anyone who's interested in thinking about how meaning is embedded in language.
This is my favorite linguistics class I've taken so far at Harvard. If you're interested at all in semantics, I would highly recommend taking this course it's not too much work per week, and the material is fun and digestible!
This is a great course and it is structured in a way that gives an excellent overview of semantics while being able to dive a little bit into each aspect that the course covered! I enjoyed this class a lot.
This class is super interesting and definitely worth taking! Lectures and problem sets are designed so that if you're new to the material, you'll still be able to understand it well, and if you're not new to the material, you'll still find it engaging. Section allows you to engage in fun debates with classmates where you may find yourself questioning whether a sentence truly "means" what you thought it did. The material is very eye–opening, and I'd recommend it whether you're majoring in Linguistics or in another field. For instance, I'm majoring in CS and there are a bunch of highly relevant topics covered in this course such as lambda calculus. This class covers a bunch of big concepts such as set theory and predicate logic, but does so in a very chill and approachable way so you won't feel too lost or confused. I highly recommend taking it! (Also by the end of the class you'll be able to remember the names of the band members of ABBA by heart!)
