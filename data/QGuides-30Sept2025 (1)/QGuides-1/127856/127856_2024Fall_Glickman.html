This was a fun and pretty chill class!

The course covers linear models and generalized linear models. The first half discusses linear models, OLS regression, the hat matrix and various properties, the Gauss–Markov theorem, likelihood–ratio tests on nested models, confidence intervals, etc. There is a lecture on linear–algebra techniques, which was definitely good; I think this could be made into a pset0, however, which would save some time. The second half discusses GLMs and the analogs of all the linear model topics for GLMs. Then quasi–likelihood models, overdispersion (based on setting as unknown dispersion parameter of an exponential family), and model selection (AIC/BIC/etc.) are discussed. The material is pretty good and relatively fundamental for understanding linear models. It gives good familiarity with linear algebra as well. The GLM material is also useful and explained matter–of–factly.

Workload–wise, the course is pretty chill; there are 6 HWs. I actually really like the HW structure, which often involves checking things stated in lecture. They are somewhat computationally intensive, but I think this is really effective at helping us review the material. Overall, a good class to learn theory behind linear models/GLMs, which is very important for much of modeling!

Note: there are no programming exercises. If you want familiarity with actually implementing models, take the undergrad versions 139/149. I do think 244 could maybe cover more implementation exercises, but it is marketed as a theory–based course, so this is fine.
