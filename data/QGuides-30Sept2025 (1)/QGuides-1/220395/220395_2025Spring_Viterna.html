I really enjoyed this class and highly suggest it. I will say, sometimes it feels like a high school class (mandatory attendance with no exceptions – no phones etc.), but I think it was effective in getting people to participate more. There wasn't a class I felt like i didn't learn, or that I felt dragged on. I appreciated how faced faced it was, and if you figure out how to more effectively read papers the assigned reading isn't as bad as it may seem. This class really changed the way I think about the world, and I appreciate how they brought real world topics into discussion, it made me feel a lot more connected to what was going on around me. I would take this class a thousand times again and if you have any questions about the why behind the bigger issues affecting us then this class will make you more equipped to keep asking and understanding those questions. BE PREPARED this class has a lot of lectures that go very fast so find ways to keep up. Overall, this class if fairly material heavy.
I don't know if Clayton is going to TF again, but he was amazing. The assignments for the class aren't bad. Professor Viterna's weeks, the quizzes were harder for some reason and I felt like I was tricked somehow, but Professor Ciocca Eller's were really good if you did the readings and paid attention.
The class is tremendously interesting and the lectures are always engaging. The coursework is very manageable and the teaching team is incredibly supportive. All in all, a great course that you should definitely considering taking!
