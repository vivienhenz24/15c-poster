The information taught in this class is quite useful and it is not excessively difficult but be aware that there is a lot of work to do. Every week, you are responsible for either a problem set or a quiz (although these often felt more like exams), and 2 pre–class assignments as well as attendance in class on Monday and Wednesday mornings. In addition to this, there are two group assignments that you are given a relatively short window to complete. The course load is not unmanagable (especially if you are comfortable with exam–taking) but it is certainly nontrivial. In addition, the grading policy, while it does give greater flexibility, can be stressful as it can make it difficult to understand how you are doing in the course. The group projects in particular are implemented as to only bring your grade down upon being unsatisfactory and not increase your grade if they are satisfactory.

This class is structured in the flipped classroom model, which means that you are expected to teach yourself the material for the upcoming class and then work on problems with your peers during class. I'm sure this worked well for some students, but it was not so helpful for me. Much of the time, Margo would see that many students were confused about the same topic in the pre–class discussions, and go over the material at the beginning of the class. She would also walk through the in–class problems at the end of class; I found these explanations from Margo magnitudes more helpful than going through readings and problems on my own. So, just be aware the way this class is structured and be sure that the flipped classroom won't be an issue. Other than that, excellent course, and Margo is amazing!

If you're interested in taking an applied course, I would say this is a really great introduction! You learn how to translate problems into mathematical models and statements, especially through the EO projects. If you want a class that is going to be really solidly grounded in linear algebra or mathematical theory, this class is not for you. My main dissatisfaction with the course was that we didn't go over any of the math motivating the techniques or algorithms we were running. I wished we spent more time going through the linear algebra rather than doing things for the sake of doing things. TLDR, this course is great for the very applied focused applied mathematicians, whereas if you want more of a focus on mathematical theory or why we do the things we do, you're probably not going to be satisfied by this class.
Definitely take this course if you have any interest in mathematical problem solving.
