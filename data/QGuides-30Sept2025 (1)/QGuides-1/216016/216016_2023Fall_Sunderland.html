This class is a chill but useful way to learn about everything from fieldwork to data analysis to scientific writing. If that is something that might be useful in the future take this class! Otherwise, you probably won't learn that much chemistry or other theoretical stuff from it.
Take this class if you are interested in the topic.
The strengths of the course was when we got to conduct the field experiments, especially the second time we went to the cape and got to do the water table charting and also looking at the dissolved oxygen in the water. This course is somewhat disorganized but thats just because it is based on real research which is not always streamlined. We never got grades, but we did get feedback on our submission. Most work is done with a group. I think because everyone is so enthusiastic about the class that everyone pulled their weight in the groups. A background in research writing would be good to have because that is basically the whole course and during the data analysis portion, it would be good to know how to use python, but the class seemed to be manageable even if one does not have prior experience. I learned a lot of water pollution testing techniques and how to approach the MeHg problem in the local region. It made me feel closer to a potential career because I originally felt limited about what I could do as an EPS concentrator, but this class opened up a lot of perspectives for me.
Take it! Professor Sunderland is awesome and you'll learn a lot about the steps to writing research into a report. A really great methods course.
This class is very hands–on and is a great introduction to scientific research. There was a heavy emphasis on the writing of a scientific paper, more so than on the labwork itself. The structure of the class is unique, and you will spend all semester working in the same group. This wasn't my favorite ESE class that I've ever taken, but it was good. Plus, Elsie is an amazing instructor!
This is a fantastic class, especially for anyone even potentially interested in a career in environmental science. The class is very real–world work focused, which makes all of the assignments and work super interesting and relevant. Also, there is a big component of the class which allows students to learn about careers in the field of environmental science, which is really insightful!
TAKE THIS CLASS IF YOU CAN!!! By far the most fun class I've taken at Harvard (Elsie told us on the first day that her teaching philosophy centers around fun). This class gave me a really thorough sense of what it is like to conduct scientific research in this field. I absolutely loved how this course was structured to give students an opportunity to learn about how to come up with a research project, how to conduct field work, how to conduct lab work, how to analyze data, and how to give a scientific presentation. I felt like I really got a sense of what doing research was like and it was super fun! Although this class is super chill and doesn't have a lot of work, I still got so much out of it.
