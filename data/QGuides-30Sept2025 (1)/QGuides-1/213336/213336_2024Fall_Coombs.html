Take this class!! Susan and Garth are amazing professors, the workload is manageable and interesting. Great class if you actually want to learn how to apply programming in psychology.
This is a FANTASTIC intro class! Having two professors in a small–format coding course made it very easy to get help and to consistently have an understanding of what was going on. The workload was very manageable, and this is truly and introduction class if you have no experience!!
Definitely take this if you want to learn programming in a digestible way and are scared of the CS department.
This course is an excellent introduction to JavaScript and R. I knew no JS before this semester, but now, I feel like I have a strong foundation (though, of course, I still have much to learn). I was already very familiar with R, but I think that if I weren't, I would have gained a lot from that part of the course, too. I would highly recommend this class if you're interested in learning how to program and run experiments online.
This class is great for psychology concentrators who want to learn more about coding! I previously had none/very little knowledge on how coding could help with psych experiments or work, and this class definitely made it feel much more accessible. Both Susan and Garth are also super great at answering questions and helping you understand the material, and both were super accessible for office hours or to help with task sets.
TAKE THE COURSE! Susan and Garth are wonderful professors and are super flexible and understanding. They genuinely just want to make the course as helpful and useful for you and your educational goals as possible. Note that this is not a data analysis course, although there are some elements of that in some weeks, and this is also not a research methods or writing course. If you want to learn the basics of how to build a website and use it to run a study, and then learn the basics of how to analyze the data you collected and visualize it, this is for you. Honestly, I think the syllabus was a little scarier than the course and the course was really chill.
