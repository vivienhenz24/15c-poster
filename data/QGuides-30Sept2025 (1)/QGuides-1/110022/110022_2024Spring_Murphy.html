Susan is an amazing professor and this class really brings you back to the fundamentals of causal inference that are often overlooked/ignored when taking more applied courses from other departments. This is definitely a great chance to learn more about statistics and the field of causal inference with a great professor! Just don't come in thinking it will be heavily applied, as it tends to be more on the theoretical side, with quite a bit of math, but not in a bad way.
Overall, I think this course is a chiller stat class in which you still learn useful concepts and tools. Here are some unstructured thoughts:
– Susan is a passionate and caring instructor, but lectures did get a bit tedious at times (e.g. getting bogged down in calculations/derivations)
– Nathan was an amazing TF –– I wish there was an optional section for this class!
– HW was biweekly and not overly long, but staying on top of conceptual check quizzes before each class could be annoying (esp since they only opened 24 hours before class)
– the midterm leaned very conceptual, while the final was much more math heavy
– reading the book and going through the lecture slides were the most helpful for exam prep
This course focuses on the mathematical foundation of early topics in causal inference and less on the application.

This class is very math heavy but the exam is very fair and focuses on conceptual understanding.

A large chunk of class time is spent on deriving complicated proof –– the result of each proof is very consequential to the material but I think professor Murphy could have also just pre–write the proof and assign students to review rather than spending class time performing complicated live derivation.

I'd prefer it if the course can cover more materials such as more modern topics in causal inference.

I'd recommend this class to students with strong statistics background otherwise it can be very easy to fall behind, even with some prior basic knowledge in causal inference.
Very good teaching, useful materials, good combination of theory and application. Recommend!
This course is a great introduction to causal inference. It focuses important conceptual idea, namely introducing the Potential Outcomes framework and how this can be used to identify causal effects. It starts with the randomized experiment case which allows for very simplified inference of causal effects. It then spends much of the course on Observational Causal inference which is an important set of tools and ideas for analyzing data in situations where data is given, not obtained from a randomized trial; the more common situation.

The course assignments are focused on simulation of data followed by interpretation of that data in a way that a scientist would interpret that data but since we know what the origin of the data is, we are able to understand in what conditions and why the casual analysis may fail or succeed. I highly recommend this course for anyone considering a data scientist role in Tech as many of these roles work on A/B testing for which this class provides the statistical knowledge.
Very thought–provoking class! It's about developing causal thinking, and every single lecture left me inspired. Susan is an amazing teacher!
