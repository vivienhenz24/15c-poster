This seminar is very nice but it requires a good amount of work, roughly 5–6 hours a week if you do the readings. I would want to emphasize that it is a art history and architecture, as said in the course description; if you want to get better introduced into this sphere of work, I would recommend the course. Professor Khandelwal is also very nice and engaged with his students.
Take this only if you think you can get some kind of concentration credit for it.
Class is interesting but really deep dives and final project is a lot of work.
Honestly, I wouldn't really recommend this course. I feel like a lot of the work in the class wasn't very rewarding. The readings were way too long every week. There weren't really any big assignments besides the journal entries every week, which I really liked because they were a chance to free write about the readings and the topics in the readings. The actual class often got very boring and felt way too long. It wasn't very engaging for being such a small class, which isn't the best combination. The final project was also a lot of work for a seminar.
